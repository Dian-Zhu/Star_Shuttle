# SSH 远程终端工具 - 前端设计规范

## 1. 概述
本设计规范旨在为 SSH 远程终端工具提供统一的视觉语言和交互体验。设计风格采用现代、简洁的暗黑主题，专注于开发者效率和沉浸式体验。

## 2. 色彩系统 (Color Palette)

基于 Tailwind CSS Slate 色系，提供舒适的对比度。

### 背景色
- **主背景 (Main Background)**: `slate-950` (#020617) - 用于终端背景和应用底色。
- **侧边栏/面板 (Sidebar/Panel)**: `slate-900` (#0f172a) - 用于侧边栏、头部和模态框。
- **表面/卡片 (Surface/Card)**: `slate-800` (#1e293b) - 用于列表项 hover、输入框边框。

### 文本色
- **主要文本 (Primary Text)**: `slate-200` (#e2e8f0) - 用于主要内容。
- **次要文本 (Secondary Text)**: `slate-400` (#94a3b8) - 用于元数据、标签和说明。
- **高亮文本 (Highlight Text)**: `white` (#ffffff) - 用于强调和选中状态。

### 功能色
- **主要品牌色 (Primary Brand)**: `blue-600` (#2563eb) - 用于按钮、激活状态。
- **成功 (Success)**: `green-500` (#22c55e) - 用于成功提示。
- **错误 (Error)**: `red-500` (#ef4444) - 用于错误提示和删除操作。

## 3. 字体系统 (Typography)

### 字体家族
- **UI 字体**: System UI (`-apple-system`, `BlinkMacSystemFont`, `Segoe UI`, `Roboto`, etc.)
- **代码/终端字体**: `JetBrains Mono`, `Consolas`, `Monaco`, `monospace`

- **Small (说明)**: 0.75rem (12px), regular

## 4. 组件规范 (Components)

### 按钮 (Buttons)
- **主要按钮**: 蓝色背景，白色文字，圆角 `rounded-lg`，阴影 `shadow-lg`。
- **次要按钮**: 透明或深色背景，灰色文字，hover 时变亮。
- **危险按钮**: 红色文字或背景。

### 输入框 (Inputs)
- 背景: `slate-950`
- 边框: `slate-700` (focus: `blue-500`)
- 圆角: `rounded-lg`
- 内边距: `px-3 py-2`

### 侧边栏 (Sidebar)
- 宽度: `w-64` (256px)
- 边框: 右侧 `border-r border-slate-800`
- 列表项: 高度适中，hover 时背景色变化。

### 终端窗口 (Terminal)
- 占据剩余空间
- 无滚动条 (使用 xterm 内部滚动)
- 标签页栏位于顶部，支持关闭和切换。

## 5. 交互规范 (Interaction)

- **过渡动画**: 所有可交互元素 (hover, focus) 应有 `transition-all duration-150`。
- **模态框**: 使用背景模糊 `backdrop-blur-sm` 和淡入淡出动画。
- **反馈**: 操作成功或失败时，右上角显示 Toast 提示。

## 6. 图标系统 (Icons)
使用简洁的线性图标 (Stroke width 2px)，大小通常为 `w-4 h-4` 或 `w-5 h-5`。
