# 功能需求补充建议

基于当前项目现状（Tauri 2 + Svelte + Rust）及已实现功能分析，以下是对 SSH 客户端功能的补充建议，旨在提升工具的专业性、易用性及安全性。

## 1. 核心功能模块的完整性评估

*   **端口转发与隧道 (Port Forwarding & Tunneling)**
    *   **现状**: 目前仅提及 "SSH 代理转发 (Agent Forwarding)"。
    *   **建议**: 补充完整的 SSH 隧道支持，包括：
        *   **本地端口转发 (Local / -L)**: 将本地端口流量转发至远程目标。
        *   **远程端口转发 (Remote / -R)**: 将远程端口流量转发至本地目标。
        *   **动态端口转发 (Dynamic / -D)**: 创建本地 SOCKS 代理，实现动态流量转发。
    *   **价值**: 运维人员核心需求，用于穿透防火墙、访问内网服务或调试远程数据库。

*   **连接配置导入/导出 (Import/Export)**
    *   **现状**: 仅支持 CRUD，无迁移方案。
    *   **建议**:
        *   支持导出连接配置为 JSON/YAML 或加密文件（便于备份）。
        *   支持导入功能，兼容主流客户端（如 PuTTY, OpenSSH config, FinalShell, Xshell 等）的格式。
    *   **价值**: 降低用户迁移成本，防止数据丢失。

*   **代理连接支持 (Proxy Jump / Proxy Command)**
    *   **现状**: 未提及。
    *   **建议**: 支持为 SSH 连接配置前置代理（SOCKS5, HTTP, 或 Jump Host/Bastion Host 跳板机）。
    *   **价值**: 解决复杂网络环境下的连接问题（如需通过 VPN 或跳板机访问服务器）。

*   **快捷命令/片段 (Snippets/Macros)**
    *   **现状**: 仅有基础终端交互。
    *   **建议**: 增加“命令片段库”功能，允许用户保存常用脚本或命令（支持参数占位符），一键发送至终端。
    *   **价值**: 提高重复性运维工作的效率。

## 2. 用户交互体验优化 (UX)

*   **终端搜索 (Find in Terminal)**
    *   **现状**: 依赖 xterm.js，但未提及搜索插件。
    *   **建议**: 集成 `xterm-addon-search`，支持 `Ctrl+F` 在终端历史输出中查找关键字，并支持正则匹配和高亮显示。
    *   **价值**: 快速定位日志报错或特定配置项。

*   **命令面板 (Command Palette)**
    *   **建议**: 引入类似 VS Code 的 `Ctrl+Shift+P` 命令面板，支持模糊搜索连接、执行特定命令、切换主题等。
    *   **价值**: 提升键盘党（Power User）的操作效率，减少鼠标点击。

*   **多路复用/广播输入 (Multi-Execution)**
    *   **建议**: 允许用户同时选中多个终端 Tab，在一个终端输入命令时，实时广播到所有选中的终端。
    *   **价值**: 批量维护多台同构服务器（如批量更新、重启服务）。

*   **外部编辑器集成**
    *   **建议**: 在 SFTP 文件列表中，支持“使用外部编辑器打开”（如 VS Code, Sublime Text），编辑保存后自动同步上传。
    *   **价值**: 解决终端内 Vim/Nano 编辑复杂文件的体验瓶颈。

## 3. 系统性能或安全性方面的增强

*   **交互式认证优化 (MFA/2FA Support)**
    *   **现状**: 支持密码/密钥。
    *   **建议**: 针对开启了 MFA（如 Google Authenticator, Duo）的服务器，优化 `Keyboard-Interactive` 认证流程，提供清晰的验证码输入弹窗或行内提示。

*   **应用级安全锁 (App Lock)**
    *   **建议**: 增加“离开自动锁定”功能。当应用后台运行或无操作超过 N 分钟，自动模糊界面并要求输入主密码解锁。
    *   **价值**: 防止离开工位时的未授权访问。

*   **终端渲染性能 (WebGL)**
    *   **建议**: 确保集成 `xterm-addon-webgl`。
    *   **价值**: 在 `cat` 查看大文件或产生大量日志输出时，大幅降低 CPU 占用率，保持界面流畅。

## 4. 数据分析或报表功能

*   **连接统计与审计**
    *   **建议**:
        *   **仪表盘**: 展示“最常访问主机”、“最近连接时间分布”。
        *   **流量监控**: 实时显示当前会话的上传/下载速率。
        *   **操作审计**: 记录关键高危指令（如 `rm -rf`, `reboot`）的执行时间与对象（需配合后端 Shell 解析或简单的正则匹配）。

## 5. 与其他系统的集成

*   **配置云同步 (Cloud Sync)**
    *   **建议**: 支持通过 Gist (GitHub), WebDAV, 或 S3 存储桶同步加密后的配置文件。
    *   **价值**: 实现多设备（家里/公司）间的工作环境无缝切换。

*   **本地 Shell 集成**
    *   **建议**: 在终端 Tab 中不仅支持 SSH，也支持打开本地 Shell (PowerShell, WSL, Bash, Zsh)。
    *   **价值**: 统一本地与远程的开发体验，无需切换终端软件。
